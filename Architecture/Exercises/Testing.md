# Übung Testing

## Kompetenzen

Die Studierenden können mit automatischen Tests sicherstellen, dass die entwickelte Applikation korrekt und zuverlässig
funktioniert.

## Aufgabe

Untersuche folgende Fragen:

- Wie kann eure Applikation automatisch getestet werden? 
- Sind dafür Anpassungen an der Struktur nötig?
- Welche Tests sind aufwendig umzusetzen? Welche sind einfach?
- Gibt es z.B. Möglichkeiten, mit wenig Zeilen Code gewisse Funktionalität zu testen?

## Siehe auch
