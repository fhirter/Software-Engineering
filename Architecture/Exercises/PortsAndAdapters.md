# Übung Ports & Adapters Architektur

Definiert in eurer Applikation die Schnittstellen (z.B. REST API, Datenbank oder Messaging System) und definiert dafür
Ports und Adapter.

Skizziert und Dokumentiert die Architektur mit einem geeigneten Werkzeug (z.B. diagrams.net, PlantUML, Mermaid, Papier).

Setzt diese Architektur in Code um.
